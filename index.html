Firebase Sync
Free setup (~5 min):
1. console.firebase.google.com → New project
2. Build → Realtime Database → Create database
3. Choose Start in test mode → any region
4. Copy the URL shown (ends in .firebaseio.com)
5. Paste above and click Connect

Security note: test mode allows all reads/writes for 30 days. To lock it down to your team only, set a simple secret in Database Rules:
{ "rules": { ".read": true, ".write": true } }
